# Typescript cli Template

## Install

```shell
npm install @chepe/ts-cli-template
```

## Usage

**List directories**

```shell
ts-cli-template -ls
ts-cli-template --ls ~/
```

**Check the Weather**

```shell
ts-cli-template --weather
```

**Check Version**

```shell
ts-cli-template -V
```

**Help**

```shell
ts-cli-template -h
```
